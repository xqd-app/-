# 数据集描述

本项目使用的数据集主要包括社交网络中的用户关系数据。数据集的详细信息如下：

## 数据来源

数据集来源于 [数据源名称]，包含用户之间的互动记录。

## 数据格式

数据集以 [数据格式，如 CSV、JSON 等] 格式存储，主要字段包括：

- `user_id`: 用户的唯一标识符
- `friend_id`: 朋友的唯一标识符
- `interaction_type`: 用户之间的互动类型（如关注、点赞、评论等）
- `timestamp`: 互动发生的时间戳

## 数据集结构

数据集的目录结构如下：

```
data/
├── raw/
│   └── social_network_data.csv
└── processed/
    └── cleaned_social_network_data.csv
```

## 数据预处理

在使用数据集之前，需进行以下预处理步骤：

1. 数据清洗：去除重复记录和缺失值。
2. 数据转换：将时间戳转换为可读格式。
3. 数据归一化：对用户互动类型进行编码。

## 使用说明

在项目中，数据集将用于社交关系的可视化分析和网络分析。请确保在运行分析脚本之前，已正确加载和处理数据集。
